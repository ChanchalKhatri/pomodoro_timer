# ⏳ Pomodoro Timer

A modern and responsive Pomodoro Timer built with **React (Vite)** to help improve focus and productivity using the Pomodoro technique.

## 🚀 Features

* 25-minute focus timer
* 5-minute break timer
* Start, pause, and reset controls
* Circular progress indicator
* Session counter
* LocalStorage support
* Responsive design
* Smooth animations with Framer Motion

## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* Framer Motion
* Lucide React
* LocalStorage

## 📦 Installation

```bash
npm install
npm run dev
```
